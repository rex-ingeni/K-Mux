# K-Mux

Claude Code / Codex CLI를 감싸는 macOS 데스크톱 앱. 각 CLI의 구조화 프로토콜을
파싱해 스트리밍 채팅 · 마크다운/링크 미리보기 · 툴 카드 · 컨텍스트 게이지로 보여주고,
터미널(TUI) 탭과 GUI 탭을 세션별로 오갑니다.

## 다운로드 (Apple Silicon / macOS)

[**Releases**](https://github.com/rex-ingeni/K-Mux/releases)에서 최신
`K-Mux-<버전>-arm64.zip`을 받아 압축을 풀고 `K-Mux.app`을 실행하세요.

설치 후에는 앱 좌측 하단 **업데이트 확인** 버튼(또는 메뉴바 트레이 → *업데이트 하기*)으로
새 버전이 올라오면 **자동으로 다운로드·교체·재실행**됩니다.
