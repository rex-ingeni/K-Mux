터미널 그대로,
앱처럼 편하게.

GUI는 CLI를 흉내 내지 않습니다 — 진짜 CLI는 그대로 두고, 같은 프로토콜을 파싱해 보여줍니다.

01 · 터미널과 GUI, 나란히

터미널에서 쓰던 기능을
GUI에서 그대로.

터미널 탭은 진짜 CLI(실 PTY)를 그대로 돌리고, GUI 탭은 그 CLI가 말하는 같은 스트림 프로토콜을 파싱해 그린 채팅입니다. 그래서 슬래시 커맨드·권한 모드·모델 선택·/compact까지 GUI에서 그대로 동작합니다. 한 세션에 두 탭이 함께 있고, 각 탭은 같은 폴더 위에서 도는 서로 독립된 대화입니다.

💬 스트리밍 · 툴 카드 · 컨텍스트 압축 표시 · 권한 모드/모델 선택
02 · 내 눈에 맞게

테마와 글씨 크기를
취향대로.

라이트 · 다크 · 시스템 테마에 14가지 강조 색상을 스와치로 고릅니다. GUI와 터미널 커서 색까지 함께 바뀝니다. 글씨 크기는 ⌘+ / ⌘− / ⌘0 이나 트랙패드 핀치 줌으로 GUI·터미널 모두 조절할 수 있습니다.

Get started

설치하고,
바로 시작하세요.

  1. 최신 DMG를 다운로드합니다릴리스 페이지에서 K-Mux-<버전>-arm64.dmg를 받습니다. (Apple Silicon)
  2. Applications 폴더로 옮깁니다DMG를 열고 K-Mux.app을 Applications 폴더로 드래그합니다.
  3. 처음 한 번 열기를 허용합니다“확인되지 않은 개발자” 경고가 뜨면 우클릭 → 열기로 한 번만 허용합니다.
  4. Claude Code / Codex를 준비합니다둘 중 하나 이상 CLI가 설치돼 있으면 세션을 열 수 있습니다.
✅

업데이트는 원클릭으로

새 버전을 자동으로 감지해 좌측 하단 버튼과 메뉴바에 알림을 띄웁니다. 그 버튼을 누르면 다운로드 → 무결성 검증 → 교체 → 재실행까지 한 번에 진행됩니다(설치는 눌러야 시작). 함께 제공되는 .app.zip은 업데이트용이라 직접 받을 필요가 없습니다.
