K-Mux는 두 CLI가 내보내는 구조화된 스트림을 파싱해, 터미널 로그가 아니라 앱 스타일 채팅으로 보여줍니다. 터미널과 GUI를 오가고, 라이트·다크와 색상 테마로 내 눈에 맞게 꾸밉니다.

See it in use

터미널 그대로, 앱처럼 편하게.

앱 화면은 CLI를 흉내 내지 않습니다 — 진짜 CLI는 그대로 두고, 같은 프로토콜을 파싱해 보여줍니다.

01 · 터미널과 GUI, 나란히

터미널에서 쓰던 기능을 그대로.

터미널 탭은 진짜 CLI를 그대로 돌리고, GUI 탭은 그 CLI가 말하는 같은 스트림 프로토콜을 파싱해 그린 채팅입니다. 그래서 슬래시 커맨드·권한 모드·모델 선택·/compact까지 앱 화면에서 그대로 동작합니다. 한 세션에 두 탭이 함께 있고, 각 탭은 같은 폴더 위에서 도는 서로 독립된 대화입니다.

스트리밍 · 툴 카드 · 컨텍스트 압축 표시 · 권한 모드/모델 선택
02 · 여러 세션을 한 화면에

필요한 만큼 나눠서 나란히.

세션을 분할 타일로 배치하고 경계를 드래그해 크기를 조절합니다. 상단바를 끌어 위치를 바꾸거나 나누고, 그룹으로 묶어 관리할 수 있습니다. Claude·Codex·셸을 한 화면에 나란히 두세요.

분할 타일 · 그룹 · 드래그로 재배치

Editor-grade input

터미널 입력창인데, 에디터처럼.

CLI의 입력줄을 그대로 쓰면서, macOS 텍스트 필드 같은 편집을 얹었습니다. 한글 입력(IME)에서도 안전하게 동작합니다.

마우스로 선택·이동클릭해서 커서를 옮기고, 드래그로 블록을 선택해 잘라내거나 지웁니다.
단어·줄 단위 이동⌥+←/→ 단어 단위, ⌘+←/→ 줄 처음·끝으로.
실행 취소 · 다시 실행⌘Z / ⇧⌘Z — 히스토리 방식이라 깜빡임 없이.
덮어쓰기 입력·붙여넣기블록을 선택한 채 입력하거나 붙여넣으면 그 자리에 바로 교체됩니다.
전체 선택 · 잘라내기⌘A / ⌘X — 여러 줄 입력도 정확하게.
Claude · Codex 모두테두리 있는 입력창과 없는 입력창을 자동으로 인식합니다.

Get started

설치하고, 바로 시작하세요.

  1. 최신 DMG를 다운로드합니다릴리스 페이지에서 K-Mux-<버전>-arm64.dmg를 받습니다. (Apple Silicon)
  2. Applications 폴더로 옮깁니다DMG를 열고 K-Mux.app을 Applications 폴더로 드래그합니다.
  3. 처음 한 번 열기를 허용합니다“확인되지 않은 개발자” 경고가 뜨면 우클릭 → 열기로 한 번만 허용합니다.
  4. Claude Code / Codex를 준비합니다둘 중 하나 이상 CLI가 설치돼 있으면 세션을 열 수 있습니다.

업데이트는 원클릭으로

새 버전을 자동으로 감지해 좌측 하단 버튼과 메뉴바에 알림을 띄웁니다. 그 버튼을 누르면 다운로드 → 무결성 검증 → 교체 → 재실행까지 한 번에 진행됩니다(설치는 눌러야 시작). 함께 제공되는 .app.zip은 업데이트용이라 직접 받을 필요가 없습니다.

터미널의 힘은 그대로, 이제 앱처럼 써보세요.
